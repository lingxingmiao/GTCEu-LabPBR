# GregTech PBR
## 中文
这是 [格雷科技社区版：非官方版](https://www.mcmod.cn/class/5343.html) 的 PBR 材质包。<br/>
此材质包需要安装支持LabPBR协议的光影使用。<br/>
若无光影还想体验立体效果可以下载 No_Shaders 后缀的文件。<br/>
总共483个视差纹理；总共3896个纹理。
## 日本語
これは [GregTech CE: Unofficial](https://modrinth.com/mod/gregtech-ce-unofficial) 用の PBR リソースパックです。<br/>
このリソースパックを使用するには、LabPBR プロトコルに対応したシェーダーが必要です。<br/>
シェーダーを使用していない場合でも立体効果を体験したい方は、「No_Shaders」接尾辞付きのファイルをダウンロードしてください。<br/>
視差テクスチャ合計 483 枚；テクスチャ総数 3896 枚。<br/>
## English
This is a PBR resource pack for [GregTech CE: Unofficial](https://modrinth.com/mod/gregtech-ce-unofficial).<br/>
This resource pack requires a shader that supports the LabPBR protocol.<br/>
If you'd like to experience the 3D/depth effect without shaders, please download the file with the "No_Shaders" suffix.<br/>
Total of 483 parallax textures; 3896 textures in total.
## 构建
1. 克隆仓库 https://github.com/lingxingmiao/GTCEu-LabPBR.git
2. 下载构建依赖库 [Mpbr.py](https://github.com/lingxingmiao/Tools/blob/main/Minecraft%20%E6%B3%95%E7%BA%BF%E7%BA%B9%E7%90%86%E7%94%9F%E6%88%90%E5%B7%A5%E5%85%B7/ver2.0/Mpbr%20ver2.2.py) 并手动重命名 `Mpbr.py`
3. 运行 [构建资源包.py](https://github.com/lingxingmiao/GTCEu-LabPBR/blob/main/%E6%9E%84%E5%BB%BA%E8%B5%84%E6%BA%90%E5%8C%85.py)
